RFID Card Actions

  • START - Start or restart current level
  • MOVE - Move robot forward
  • LEFT - Turn robot left
  • SUBMIT - Check if robot reached treasure

Difficulty Rule

Levels 1–5 are Easy. Levels 6–10 are Medium. Levels 11–15 are Hard. After each success, the treasure moves to a new position.
