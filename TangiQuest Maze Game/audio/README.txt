TangiQuest voice audio package

Place this audio folder beside index.html, style.css, and script.js.

connected.wav = Arduino connected. Scan the start card to begin.
start.wav = Game started. Guide the robot to the treasure.
move.wav = Robot moved forward.
left.wav = Robot turned left.
submit.wav = Submit card scanned. Checking result.
correct.wav = Correct. You found the treasure. Well done.
wrong.wav = Wrong position. Please try again.
unknown.wav = Unknown RFID card scanned.
blocked.wav = Blocked. The robot hit a wall.
level_ready.wav = Next level is ready. Scan start to begin.

Use paths like audio/start.wav in your script.js file.
